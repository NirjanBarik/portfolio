# Personal Portfolio - Frontend

This is the frontend of the Personal Portfolio project, built with **React**, **Vite**, and **Tailwind CSS**.

## 🚀 Getting Started

Follow these steps to set up and preview the project locally.

### 1. Installation

First, install the project dependencies:

```bash
npm install
```

### 2. How to Preview

You can preview the project in the following ways:

#### **Development Mode** (Recommended for editing)
Run the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Once started, open [http://localhost:5173](http://localhost:5173) in your browser.

#### **Production Preview** (Test performance)
To see how the site looks in production, build and then preview:
```bash
npm run build
npm run preview
```

## 🛠️ Tech Stack
- **React 19** - UI Framework
- **Vite** - Fast Build Tool
- **Tailwind CSS 4** - Styling
- **Framer Motion** - Smooth Animations
- **Lucide React** - Modern Icons
- **React Router** - Navigation
